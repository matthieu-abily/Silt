Silt — rough fragments before form

Drop images, arrange references, write notes, draw arrows, sketch, add stickers, and frame clusters. Use it for scripts, scenario, worldbuilding, characters, locations, visual motifs, and DoP moodboards.

v0.13.5: hotfixes Select/move reliability, keeps export modes for page and selection, keeps project-title filename suggestions, and preserves object connectors. Use Connector, click one item, then click a second item. Connectors follow the items and are removed automatically if either endpoint disappears.

iPad: use Pan to move around. Pencil/finger drawing works best with the Pen tool. Text notes and frame labels edit in-place.

Shared folders: save .siltboard files inside Google Drive, iCloud Drive, Dropbox, Syncthing, or any normal synced folder.

Shortcuts: Space-drag pan, wheel/pinch zoom where supported, ⌘Z undo, ⇧⌘Z redo, ⌘D duplicate, Delete remove, double-click/tap a note to edit, Shift-click to multi-select, drag on blank canvas to lasso, ⌘G group, ⇧⌘G ungroup, ⌘L lock/unlock. Turn Snap off in the top bar when you want completely free placement. Use Canvas → A4 or 16:9 when you want a moodboard with a fixed export ratio. Export → Board PNG exports the active A4/16:9 page exactly; in Free mode it crops to the whole used board with a small margin. Export → Area PNG lets you drag a screenshot-style export window. The Project title in the top bar becomes the suggested file name.
